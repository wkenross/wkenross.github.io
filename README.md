# wkenross.github.io

Root GitHub Pages website for Ken Rossiter and FRAME Sales Automation.

Target public URL:

```text
https://wkenross.github.io/
```

This root site is intended for forms that require a company or personal website
root URL. It links onward to the FRAME showcase project site:

```text
https://wkenross.github.io/frame-showcase/
```

## Publish

Create a GitHub repository named exactly:

```text
wkenross.github.io
```

Push this folder to the repository's `main` branch. GitHub Pages user sites are
served from the repository root.
