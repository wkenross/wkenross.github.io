Install FRAME Automation

Purpose

This page introduces the installation lane for the FRAME automation framework. It is intended for reviewers who want to understand how the Windows 11 laptop, VS Code workspaces, OneDrive project folders, Edge browser automation, and FRAME scripts fit together.

What The Install Workspace Does

The install workspace organizes the machine setup and follow-on configuration needed to run FRAME workflows on a new laptop.

It covers:

  • Windows 11 workstation preparation
  • PowerShell and Python setup
  • VS Code workspace launch configuration
  • OneDrive project synchronization
  • Edge browser profile and WebDriver support
  • FRAME folder placement for local and OneDrive modes
  • publication support for GitHub Pages and the FRAME showcase
