Safety And Review

Purpose

Keep the web workflow useful without exposing private operational details.

Blocked By Default

  • credentials and tokens
  • browser profiles
  • Outlook account state
  • live CRM/Jira tenant URLs
  • workbook data
  • .pbix files with embedded private data
  • screenshots with account names
  • private customer, contact, or employer labels

Review Levels

Status Meaning
public safe for broad publishing
limited-review safe for a small reviewer group, not public yet
internal useful inside the workstation only
blocked do not publish or transform

Done Criteria

  • source catalog has a review status
  • generated pages have no private references
  • links work locally
  • certification claims are truthful
  • output can be regenerated from reviewed Markdown
