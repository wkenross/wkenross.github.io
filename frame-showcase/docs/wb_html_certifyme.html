WB HTML CertifyMe

Updated: 2026-06-09

Purpose

Use the wb_html workflow as evidence for web publishing, documentation, accessibility, and static-site delivery skills.

This is not the primary certification lane. SQL uses DP-900 and PL-300. Agile uses PSM I. wb_html packages the evidence and can support web-adjacent learning goals.

Practical Credential Signals

Potential future credentials or learning badges:

  • GitHub Foundations or GitHub Pages learning
  • Microsoft Learn modules on Power BI report publishing and dashboard sharing
  • accessibility fundamentals
  • HTML/CSS fundamentals
  • static-site documentation tooling such as MkDocs or Docusaurus

No credential should be listed as earned until it is earned.

Evidence To Build

  • local static site generated from Markdown
  • GitHub Pages-ready output folder
  • style guide with repeatable page kinds
  • privacy and publication review checklist
  • SQL/Power BI page using synthetic data
  • Agile/Jira page using scrubbed board/report language
  • CRM page using generic HubSpot/Salesforce wording
