How FRAME works

Documented enough to hand off. Scripted enough to repeat.

FRAME uses a small set of durable artifacts: PlanMe for direction, CodeMe for implementation principles, BuildMe for setup, TrainMe for learning, and ObfuscateMe for publishing safely.

The public site stays review-first. Private customer, employer, workbook, email, and browser-profile details are removed or generalized before publication.
